# Kurulum ve Çalıştırma

## Kurulum
1. `backend/DietSocial.API` dizinine gidin:
   ```bash
   cd backend/DietSocial.API
   ```
2. NuGet paketlerini yükleyin:
   ```bash
   dotnet restore
   ```

## Çalıştırma
Projeyi başlatmak için:
```bash
   dotnet run
```

Swagger arayüzü için: http://localhost:5000/swagger
